☀️ *Morning Brief* — Tuesday, Jul 21

*The trust layer for agent capabilities just became the story.* Island's security team found ~7,600 malicious GitHub repos — 800+ posing as *AI Skills or MCP servers* — that delivered SmartLoader→StealC malware to 14M+ downloads. The new twist they call *AgentBaiting*: Claude Code, Gemini, and ChatGPT all *auto-discovered* these fake repos and handed users the install instructions, no link needed. A typosquat of `awesome-claude-skills` (one-char profile diff) shipped a poisoned ZIP. The agent capability supply chain now has the same rot npm did — and no verification layer exists yet.
