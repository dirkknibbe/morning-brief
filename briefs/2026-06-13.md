☀️ *Morning Brief* — Saturday, Jun 13

The clearest opportunity today is a single friction repeated across four unrelated launches: *agents have no control layer*. A runaway agent doesn't crash — it quietly burns the monthly budget until the bill arrives. Everyone is rebuilding the same plumbing badly, and a standard layer is missing. That gap is your kind of gap.

🔥 *Hot Signals*
• *"The 98% Problem"* survey (BeConfident Labs) puts a number on it: ~98.4% of Claude Code is *harness*, not model — context, tools, permissions, sandboxing, recovery. Capability lives a layer below the model now. Required reading.
• *LLM Control Plane* (mozilla.ai): the gateway→control-plane distinction. Gateways route; control planes decide *should this request run at all* — budget caps enforced before execution, cross-app policy, provider failover.
• *BitBoard* (YC P25) launched: agents build durable, re-runnable dashboards instead of one-off chat threads. Selling "traceable & repeatable" as the product.

😤 *Developer Friction* — where the money is
• *Agentjacking* (Tenet): a fake Sentry error event hijacks coding agents into running attacker code — Sentry MCP returns untrusted payloads as *trusted system output*. 2,388 orgs exposed, 100+ agents executed in testing. The MCP trust boundary is unsolved and scary.
• "93% of permission prompts get approved" — so prompts are *not* a real safety control. Approval fatigue is a measured failure mode.

💰 *Monetization Patterns*
• *Guardian Runtime* (PyPI, MIT): "zero-latency FinOps & security firewall for AI" — local interception of every prompt/response to stop leaks + runaway tokens. Open core, governance as the paid layer.
• Same playbook three ways today: govern spend, govern data egress, govern tool calls. All recurring, all infra.

🛠️ *Meta-Agents, Skills & MCP Tools*
• *bytedance/deer-flow* — long-horizon "SuperAgent" harness (sandboxes, memory, skills, subagents, message gateway). Bytedance is now shipping an outer harness openly.
• *HKUDS/nanobot* and *zhayujie/CowAgent* — lightweight self-evolving agent harnesses gaining traction. The harness-as-product thesis is crowding fast.

📈 *Still Trending*
• Meta-agent harnesses — 3rd straight appearance (deer-flow, CowAgent, nanobot, obra/superpowers).
• Agent cost observability — control-plane + FinOps cluster building all week.

💡 *Opportunity Sparks*
• *MCP trust proxy*: a UIPE-adjacent MCP shim that tags tool outputs as untrusted and gates side-effecting calls — directly answers Agentjacking. Per-seat or per-call.
• *Budget guardrail MCP*: hard token/cost ceiling enforced *before* a sub-agent spawns. Micropayment-metered = your MCPAASTA rail.
• *"Re-runnable agent artifact" store*: BitBoard's idea, generalized beyond dashboards.

*One action today:* Read the 98% Problem survey, then sketch where UIPE's temporal-perception layer slots into a control plane — perception is the missing input to "should this request run." That's your wedge into the layer everyone's now buying.
