☀️ *Morning Brief* — Sunday, Jul 12

The sharpest signal today is *agent build-vs-buy* becoming its own product category. **OpenBenchmarks** (Show HN) publishes verified, ground-truth benchmarks *for agents to consume* — precision@100, time-to-first-audio, cost-per-correct-answer — so an agent can decide which SaaS API to call. "Real data goes in, only aggregate provider scores come out." That's the missing trust layer above MCP: discovery is solved, *selection* isn't. Directly adjacent to your MCPAASTA thesis.

🔥 *Hot Signals*
• *Mesh LLM on iroh* (265 pts) — pool your own GPUs into one OpenAI-compatible endpoint at localhost:9337. Framed explicitly as escaping metered-API "surrender."
• *AgentTransfer* — every agent gets an email address, folder, inbox; 5GB sha256-verified transfers, signed receipts, MCP server built in. One Go binary, MIT.
• *Aerial* — single Rust binary for local agent-to-agent messaging: daemon, durable JSONL mailboxes, supervisor, MCP adapter. No broker, no cloud.

😤 *Developer Friction*
• "AI coding agents read your code perfectly and understand your team not at all" — the recurring complaint: agents lack org/social context, not code comprehension.
• Mesh LLM's whole pitch is friction: no control over when models change, where data goes, or the bill. People want pluggability providers won't give.

💰 *Monetization Patterns*
• *Aether* — cloud devbox for coding agents, BYO subscription, per-second metering, 0 credits while idle. Usage-metered + bring-your-own-keys is becoming the default agent-infra pricing shape.
• OpenBenchmarks bills each provider on a disclosed plan and publishes cost-per-correct — turning pricing transparency itself into the product.

🛠️ *Meta-Agents, Skills & MCP Tools*
• *Aerial* (orchestration/messaging), *attestor* (Open-AGS "Zero Trust Execution Boundary" for agents), *AgentTransfer* (agent file I/O) — all shipping the plumbing layer: identity, messaging, sandboxed execution. The harness wars are moving from "how do agents think" to "how do agents safely act."
